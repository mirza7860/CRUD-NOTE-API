User Register

Request

Endpoint: POST http://localhost:8000/api/v1/user/register

          
            {
              "userName": "MirzaSahil",
              "password": "Sahil#786",
              "email": "Sahil@gmail.com"
            }

Response

          
            {
              "status": 201,
              "response": {
                "newUser": {
                  "userName": "MirzaSahil",
                  "email": "Sahil@gmail.com",
                  "password": "Sahil#786",
                  "notes": [],
                  "_id": "6595eda8a2282260b72b255d",
